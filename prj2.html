Artist Statement

This portfolio website is designed as a digital space where my 3D work in environment and asset design can be explored in an immersive and interactive way. Rather than presenting static images, I wanted to create a site where visitors can directly engage with my models—rotating, zooming, and navigating through them in real-time. This level of interaction reflects how 3D art is meant to be experienced: dynamically, from all angles.

Built using HTML, CSS, and JavaScript, the site integrates Sketchfab and Spline embeds to bring my 3D models to life on the web. Iframes allow for seamless previews, while custom styling and a dark-themed layout help spotlight the work itself. The homepage features a floating particle planet that sets the tone for the rest of the site—one that blends creativity, interactivity, and clean presentation.

This project was made specifically for the internet because it takes full advantage of what the web can offer: accessibility, responsiveness, and real-time interaction. Unlike a traditional printed portfolio, this site adapts to different screens and allows users to move between different 3D works using a custom-built viewer switcher—turning the website into a living gallery.

Overall, Hanz3DWorld isn't just a portfolio—it's an evolving virtual space that reflects both my artistic identity and technical growth as a 3D artist.
